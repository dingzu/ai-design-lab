# GenUI 极光流动艺术头图提示词

## 基本信息
- **图片类型**：艺术概念插画（故事感）
- **设计风格**：极光渐变风格（Aurora Gradient）
- **输出路径**：/Users/dingzu/Documents/ai-design-lab/public/images/genui-hero.jpg
- **分辨率**：2K
- **宽高比**：16:9

## 完整提示词

生成一张16:9的艺术概念插画，展现探索者面对多条极光路径的故事场景。

【故事场景】
一个设计师/创造者的简约剪影，站在画面左侧1/3处，背对或侧对观众，凝视前方。他/她的面前，多条流动的极光光带从脚下或身旁升起，向远方延伸，每条光带代表一种技术路径的可能性。整个画面营造"探索者站在选择的十字路口"的叙事感。

【视觉风格 - 极光渐变】
- 深色渐变背景：从深蓝(#0f0f1e)到深紫(#1a1a2e)到接近黑色
- 流动的极光光带：青色(#00d9ff)→ 绿色(#00ff88)→ 粉色(#ff00ff)→ 橙色(#ff6b6b)
- 梦幻、神秘、充满希望的氛围
- 光影交织，类似北极光的视觉效果
- 柔和的光晕和微小粒子点缀
- 动态流动感，从左向右延伸

【构图与布局】
- **前景（左侧）**：人物剪影轮廓，简约干净，无需细节
  - 位置：画面左侧1/3处
  - 姿态：站立，略微仰视或平视前方
  - 风格：深色剪影，边缘有柔和光晕
  
- **中景（中央）**：多条极光光带流动
  - 5-7条不同颜色的光带
  - 从人物身旁或脚下升起，向远方延伸
  - 每条光带有渐变和流动感
  - 光带之间相互穿插、交织、汇聚又分散
  
- **背景（右侧/远方）**：深邃的渐变天空
  - 深蓝到深紫渐变
  - 星点或粒子点缀
  - 光带逐渐消失在远方，营造无限延伸感

【极光光带颜色分布】
光带1：青色系 (#00d9ff 到 #0099cc)
光带2：绿色系 (#00ff88 到 #00cc66)
光带3：粉紫系 (#ff00ff 到 #cc00cc)
光带4：橙红系 (#ff6b6b 到 #ff4444)
光带5：蓝紫系 (#6666ff 到 #4444cc)

【情绪与氛围】
- 好奇、探索、充满可能性
- 梦幻而不失深度
- 未来感、科技感、艺术感并存
- 略带仰视角度，增加史诗感和希望感
- 整体从暗到亮，象征从未知到探索

【视觉细节】
- 人物剪影边缘有柔和的光晕（受极光照射）
- 极光光带使用柔和渐变，避免硬边
- 光带流动方向略带曲线，避免过于直线
- 微小粒子沿着光带流动，增强动感
- 背景深邃但不完全黑，保持层次
- 整体有柔和的发光效果（bloom）
- 无任何文字、无具体图标、纯视觉艺术表达

【禁止元素】
- 禁止真实人物面部细节
- 禁止具体UI元素或图标
- 禁止文字、标签、说明性内容
- 禁止过于密集的内容
- 禁止锐利的边缘和硬阴影
- 禁止与极光不协调的颜色
- 禁止3D物体或具象图形
- 禁止将提示词、风格描述、结构说明等元数据显示在图片中
- 禁止出现"生成一张"、"【故事场景】"、"前景"等指令性文字
